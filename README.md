# 🚀 Meu Portfólio Pessoal

Site pessoal desenvolvido para apresentar meus projetos, habilidades e formas de contato.

---

## 📌 Sobre o Projeto
Este é o meu site de portfólio, criado com foco em design limpo, responsivo e moderno. Ele conta com:
- Página inicial com apresentação
- Seção de habilidades
- Galeria de projetos
- Área de contato
- Animações e efeitos visuais

---

## 🛠️ Tecnologias Utilizadas
- **HTML5** → Estrutura das páginas
- **CSS3** → Estilização, layout responsivo e efeitos visuais
- **JavaScript** → Animações e interações
- Git e GitHub → Controle de versão e hospedagem do código
